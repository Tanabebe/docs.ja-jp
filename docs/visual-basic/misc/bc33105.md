# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="88a41-103">'If' オペランドを名前付き引数にすることはできません</span><span class="sxs-lookup"><span data-stu-id="88a41-103">'If' operands cannot be named arguments</span></span>

<span data-ttu-id="88a41-104">`If` 演算子のオペランドで名前付き引数を使用することは正しくありません。</span><span class="sxs-lookup"><span data-stu-id="88a41-104">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="88a41-105">次の例では、このエラーが発生します。</span><span class="sxs-lookup"><span data-stu-id="88a41-105">The following example causes this error:</span></span>  
  
```vb  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="88a41-106">これは、次のコードに示す名前付き引数を許可する `IIf` 関数とは異なります。</span><span class="sxs-lookup"><span data-stu-id="88a41-106">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```vb  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="88a41-107">**エラー ID:** BC33105</span><span class="sxs-lookup"><span data-stu-id="88a41-107">**Error ID:** BC33105</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="88a41-108">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="88a41-108">To correct this error</span></span>  
  
- <span data-ttu-id="88a41-109">次のコード例に示すように、オペランドから名前の割り当てを削除します。</span><span class="sxs-lookup"><span data-stu-id="88a41-109">Remove the name assignments from the operands, as shown in the following code.</span></span>  
  
    ```vb  
    result = If(i > 0, "positive", "not positive")  
    ```
