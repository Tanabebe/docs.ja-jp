## <a name="example"></a><span data-ttu-id="0c12a-119">例</span><span class="sxs-lookup"><span data-stu-id="0c12a-119">Example</span></span>

```csharp
// Code with violations

// Fixed code
if (x) { }
if (M()) { }
```

```vb
' Code with violations
```
