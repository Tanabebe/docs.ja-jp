### <a name="example"></a><span data-ttu-id="67415-127">例</span><span class="sxs-lookup"><span data-stu-id="67415-127">Example</span></span>

```csharp

{
    Hello();
    static void Hello()
    {
        Console.WriteLine("Hello");
    }
}

{
    Hello();
    void Hello()
    {
        Console.WriteLine("Hello");
    }
}
```
