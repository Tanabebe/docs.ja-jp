## <a name="example"></a><span data-ttu-id="72f1a-120">例</span><span class="sxs-lookup"><span data-stu-id="72f1a-120">Example</span></span>

```csharp
static void M1() { }
static void M2()
{
    // IDE0002: 'C.M1' can be simplified to 'M1'
    C.M1();

    // Fixed code
    M1();
}
```

    ' Fixed code
    M1()
