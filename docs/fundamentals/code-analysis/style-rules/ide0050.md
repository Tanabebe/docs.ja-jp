## <a name="overview"></a><span data-ttu-id="6563f-116">概要</span><span class="sxs-lookup"><span data-stu-id="6563f-116">Overview</span></span>

<span data-ttu-id="6563f-117">この規則では、匿名型に 2 つ以上のフィールドがある場合に、[匿名型](../../../csharp/programming-guide/classes-and-structs/anonymous-types.md)よりも[タプル](../../../csharp/language-reference/builtin-types/value-tuples.md)の使用を推奨します。</span><span class="sxs-lookup"><span data-stu-id="6563f-117">This rule recommends use of [tuples](../../../csharp/language-reference/builtin-types/value-tuples.md) over [anonymous types](../../../csharp/programming-guide/classes-and-structs/anonymous-types.md), when the anonymous type has two or more fields.</span></span> <span data-ttu-id="6563f-118">この規則には、関連付けられているコード スタイル オプションはありません。</span><span class="sxs-lookup"><span data-stu-id="6563f-118">This rule has no associated code style option.</span></span>

## <a name="example"></a><span data-ttu-id="6563f-119">例</span><span class="sxs-lookup"><span data-stu-id="6563f-119">Example</span></span>

```csharp
// Code with violations
var t1 = new { a = 1, b = 2 };

// Fixed code
var t1 = (a: 1, b: 2);
```

```vb
' Code with violations
Dim t1 = New With { .a = 1, .b = 2 }

' Fixed code
Dim t1 = (a:=1, b:=2)
```
