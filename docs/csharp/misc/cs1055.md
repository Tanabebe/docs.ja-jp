# <a name="compiler-error-cs1055"></a><span data-ttu-id="42093-103">コンパイラ エラー CS1055</span><span class="sxs-lookup"><span data-stu-id="42093-103">Compiler Error CS1055</span></span>

<span data-ttu-id="42093-104">add または remove アクセサーが必要です。</span><span class="sxs-lookup"><span data-stu-id="42093-104">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="42093-105">[event](../language-reference/keywords/event.md) がフィールドとして宣言されていない場合は、 **add** と **remove** の両方のアクセサー関数を定義する必要があります。</span><span class="sxs-lookup"><span data-stu-id="42093-105">If your [event](../language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="42093-106">次の例では CS1055 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="42093-106">The following sample generates CS1055:</span></span>  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
