- <span data-ttu-id="8c3b3-109">[ターゲット フレームワーク](../../standard/frameworks.md)の値を `netcoreapp2.0` から `netcoreapp2.1` に変更します。</span><span class="sxs-lookup"><span data-stu-id="8c3b3-109">Change the [target framework](../../standard/frameworks.md) value from `netcoreapp2.0` to `netcoreapp2.1`.</span></span> <span data-ttu-id="8c3b3-110">ターゲット フレームワークは `<TargetFramework>` または `<TargetFrameworks>` 要素によって定義されます。</span><span class="sxs-lookup"><span data-stu-id="8c3b3-110">The target framework is defined by the `<TargetFramework>` or `<TargetFrameworks>` element.</span></span>

- <span data-ttu-id="8c3b3-112">.NET Core 2.1 SDK (v 2.1.300 以降) でバンドルされているツールの `<DotNetCliToolReference>` 参照を削除します。</span><span class="sxs-lookup"><span data-stu-id="8c3b3-112">Remove `<DotNetCliToolReference>` references for tools that are bundled in the .NET Core 2.1 SDK (v 2.1.300 or later).</span></span> <span data-ttu-id="8c3b3-113">これらの参照には次が含まれています。</span><span class="sxs-lookup"><span data-stu-id="8c3b3-113">These references include:</span></span>

  - `dotnet-watch`
  - <span data-ttu-id="8c3b3-114">[dotnet-user-secrets](https://github.com/dotnet/aspnetcore/blob/master/src/Tools/dotnet-user-secrets/README.md) (Microsoft.Extensions.SecretManager.Tools)</span><span class="sxs-lookup"><span data-stu-id="8c3b3-114">[dotnet-user-secrets](https://github.com/dotnet/aspnetcore/blob/master/src/Tools/dotnet-user-secrets/README.md) (Microsoft.Extensions.SecretManager.Tools)</span></span>
  - <span data-ttu-id="8c3b3-115">[dotnet-sql-cache](https://github.com/dotnet/aspnetcore/blob/master/src/Tools/dotnet-sql-cache/README.md) (Microsoft.Extensions.Caching.SqlConfig.Tools)</span><span class="sxs-lookup"><span data-stu-id="8c3b3-115">[dotnet-sql-cache](https://github.com/dotnet/aspnetcore/blob/master/src/Tools/dotnet-sql-cache/README.md) (Microsoft.Extensions.Caching.SqlConfig.Tools)</span></span>
  - <span data-ttu-id="8c3b3-116">[dotnet-ef](/ef/core/miscellaneous/cli/dotnet) (Microsoft.EntityFrameworkCore.Tools.DotNet)</span><span class="sxs-lookup"><span data-stu-id="8c3b3-116">[dotnet-ef](/ef/core/miscellaneous/cli/dotnet) (Microsoft.EntityFrameworkCore.Tools.DotNet)</span></span>
  
  <span data-ttu-id="8c3b3-117">以前の .NET Core SDK バージョンでは、ご自分のプロジェクト ファイル内でのこれらのツールのいずれかへの参照は次の例のようになります。</span><span class="sxs-lookup"><span data-stu-id="8c3b3-117">In previous .NET Core SDK versions, the reference to one of these tools in your project file looks similar to the following example:</span></span>

  ```xml

  ```

  <span data-ttu-id="8c3b3-118">このエントリは .NET Core SDK では使用されなくなったので、ご自分のプロジェクト内にこのようなバンドルされたツールのいずれかへの参照がまだ含まれている場合は、次のような警告が表示されます。</span><span class="sxs-lookup"><span data-stu-id="8c3b3-118">Since this entry isn't used by the .NET Core SDK any longer, you'll see a warning similar to the following if you still have references to one of these bundled tools in your project:</span></span>

  <span data-ttu-id="8c3b3-119">それらのツールに対する `<DotNetCliToolReference>` 参照をプロジェクト ファイルから削除することで、この問題は修正されます。</span><span class="sxs-lookup"><span data-stu-id="8c3b3-119">Removing the `<DotNetCliToolReference>` references for those tools from your project file fixes this issue.</span></span>
